Based on https://spring.io/guides/gs/rest-service/

How to run:
Maven: java -jar target/gs-rest-service-0.1.0.jar
Gradle: java -jar build/libs/gs-rest-service-0.1.0.jar

How to access the site: 
(NOTE: This jar runs in foreground with internal tomcat and serves the URL as if tomcat-war)
http://localhost:8080/greeting
 http://localhost:8080/greeting?name=User
